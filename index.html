📚 Documentação

Campos de uma Questão
codigo (Q00001) · enunciado · opcoes {A-E} · gabarito · assunto · topico · subtopico · banca · ano · dificuldade · area · concurso · comentario · texto_base · status (ativa/rascunho/desatualizada/anulada) · motivo_status · historico[] · relacionadas[] · fonte (manual/ia/importado) · provaId · atualizada_em · atualizada_por · imagem (base64) · createdAt
Fontes de Questões
Manual: Admin preenche formulário na aba Cadastrar.
IA: Geração automática via API Claude/Gemini na aba Gerar.
Importado: Via JSON ou PDF+IA na aba Importar.
Tipos de Prova/Simulado
importado: Criado via aba Importar → Prova Anterior. Aparece em Provas Anteriores.
personalizado: Montado pelo admin ou transformado de prova. Aparece em Simulados do Papiro Tático.

Campos da prova importada: nome · banca · ano · cargo · órgão · questoesIds[] · totalQuestoes · tipo · criadoEm
Exportação de Questões
Botão "Exportar Questões" no Banco de Questões (admin). Filtros: matéria, tópico, subtópico, concurso, status, ano, sem comentário, reportadas, taxa de acerto. Formatos: JSON (para reimportar) e CSV (para análise no Excel/Sheets).
Tela de Revisão de Importação
Após exportar, corrigir externamente e reimportar, abre a tela de revisão. Cada questão pode ser aprovada, editada ou ignorada. Todos os campos são editáveis incluindo assunto e tópico. Só os aprovados são salvos via updateDoc no Firestore.
Sistema de Reportes
Alunos reportam problemas via botão 🚩 no modo TEC. Admin visualiza, exporta e resolve no Painel Admin. Questões reportadas podem ser exportadas para análise externa.
Status de Questões
ativa: Aparece no pool TEC e simulados.
rascunho: Visível só para admin, fora do pool.
desatualizada: Fora do pool, admin pode adaptar/atualizar via exportação externa.
anulada: Fora do pool, versão original preservada no histórico.
Questões Relacionadas
Campo no modal de edição — informe códigos separados por vírgula (ex: Q00001, Q00002). Quando uma questão relacionada é marcada como desatualizada, o sistema exibe um alerta sugerindo revisar as demais.
HTML Rico nas Questões
O enunciado, texto base e alternativas suportam HTML completo:

Formatação de texto: <b> negrito · <i> itálico · <u> sublinhado
Quebras de linha: <br> para itens I, II, III e listas
Imagens: <img src="..."> ou upload via botão 📷 no modal de edição
Tabelas: <table> para planilhas e dados tabulares

Upload de imagem: Banco de Questões → ✏️ Editar questão → 📷 Selecionar imagem → salva em base64 comprimido automaticamente (máx 800px, 500KB).

Texto associado: Questões com texto_base exibem um botão "➕ Texto associado" colapsável antes do enunciado, igual ao TEC Concursos.
Tipos de Importação
📋 Prova Anterior: Importa questões vinculadas a uma prova real de banca. Campos obrigatórios: nome, banca, ano. Opcionais: cargo, órgão. A prova aparece em Provas Anteriores na aba Simulados.

🎯 Simulado: Importa um simulado montado externamente. O sistema vincula às questões já existentes no banco pelo código ou enunciado, sem duplicar. Ao apagar o simulado, as questões existentes ficam intactas no banco.

📚 Questões Avulsas: Importa questões direto para o banco sem criar prova ou simulado. Útil para questões de diversas fontes ou geradas por IA.

Deduplicação automática (todos os tipos):
• Mesmo código Q00001 → vincula à existente, não duplica
• Enunciado idêntico → vincula à existente, não duplica
• Enunciado similar (>85%) → alerta para revisar
• Questão nova → cria normalmente no banco
Funcionalidades dos Simulados (Admin)
👁 Revisar Questões: Abre o modo TEC carregando apenas as questões daquele simulado/prova. Permite revisar enunciados, status e gabarito um a um.
🗑 Apagar: Modal com duas opções — apagar só o simulado (questões ficam no banco) ou apagar tudo (simulado + questões vinculadas).
📄 PDF Prova: Gera prova limpa sem gabarito com identidade visual do Papiro Tático. Capa escura + questões numeradas + alternativas. Abre para impressão automaticamente.
📄 PDF Gabarito: Gera gabarito comentado com capa roxa, gabarito rápido em grade, alternativa correta destacada em verde e comentário em destaque amarelo por questão.

Contexto: Atualmente as imagens das questões são salvas em base64 direto no Firestore. Quando o plano Blaze for ativado, migrar para Firebase Storage para melhor performance e escalabilidade.

O que mudar:
• Função uploadImagemQuestao — trocar base64 por upload para Storage
• Questões existentes com base64 — script de migração para subir as imagens no Storage e atualizar as URLs
• Regras do Storage já estão prontas no código

Benefícios: Documentos menores no Firestore, carregamento mais rápido, sem limite de 1MB por documento.

📲 Compartilhamento e Funil de Conversão
Contexto: Implementar estratégia de compartilhamento de questões via Instagram com funil de conversão para assinatura.

Referência: Modelo inspirado no QConcursos — 3 níveis de acesso: Visitante → Cadastrado gratuito → Assinante.

Fluxo planejado:
1. Aluno compartilha card da questão no Instagram (sem gabarito — só enunciado + alternativas)
2. Seguidor clica no link do card
3. Cai na questão no app → tenta responder → aparece: "Crie sua conta gratuita para ver o gabarito"
4. Cadastra conta gratuita → resolve até 5 questões por dia
5. Quer mais → CTA para assinar o Papiro Tático

O que implementar:
• Card do Instagram gerado em canvas (1080×1080) — enunciado + alternativas + branding + link da questão. SEM gabarito
• Modal de compartilhamento com 2 opções: link clicável + baixar card
• Plano gratuito no sistema de códigos — sem expiração, limite de 5 questões/dia
• Controle de limite diário por usuário (salvo no Firestore)
• Tela de boas-vindas para quem chega pelo link do Instagram — destaca a questão específica
• CTA pós-resposta — exibe número de questões disponíveis + botão de assinatura
• Banco protegido — visitante sem login não acessa nenhuma questão

Decisões já tomadas: Exigir cadastro antes de responder (não liberar sem login). Limite de 5/dia no plano gratuito cria urgência natural para assinar.

🎓 Tutoriais

Como importar uma prova completa
1. Vá para a aba Importar
2. Cole o JSON das questões ou faça upload do arquivo
3. Preencha nome da prova, ano e tempo
4. Marque "Disponibilizar como Simulado" se quiser
5. Clique em Validar e depois Confirmar
Como gerar comentários em lote
1. No Banco de Questões, clique em "Exportar Questões"
2. Filtre por "Sem comentário" e exporte em JSON
3. Traga o JSON para o chat com Claude
4. Claude gera os comentários para cada questão
5. Reimporte com o botão "Importar com Revisão" e aprove questão por questão
Como revisar uma prova mensalmente
1. No Banco de Questões, clique em "Exportar Questões"
2. Filtre por matéria e/ou tópico desejado
3. Exporte em JSON e traga para o chat com Claude
4. Claude revisa: conteúdo, gabarito, atualização legal
5. Reimporte com "Importar com Revisão" e aprove as correções
Como resolver um reporte de aluno
1. No Painel Admin, acesse o painel Reportes
2. Leia o problema reportado pelo aluno
3. Clique em "Revisar Questão" para abrir direto no banco
4. Use "Alterar Status" ou "Editar" para corrigir
5. Marque o reporte como Resolvido
Como marcar uma questão como desatualizada
1. No Banco de Questões, localize a questão
2. Clique em "⚠️ Status" na questão
3. Selecione "Desatualizada" e explique o motivo
4. A questão sai do pool TEC automaticamente
5. Para adaptar/atualizar, exporte e traga para o chat com Claude
Como criar um Simulado Oficial
1. Na aba Simulados, clique em "+ Montar Simulado"
2. Preencha nome, tempo e distribuição de matérias
3. Ative o toggle 🏆 Modo Oficial
4. O sistema sugere automaticamente as datas (próximo sábado às 9h, 4h de duração, gabarito 1h após)
5. Ajuste as datas conforme necessário e clique em Gerar
6. No dia do simulado, os alunos veem o countdown até o início
7. Após todos entregarem, clique em 🔓 Liberar Gabarito no card
8. Alunos veem o resultado + ranking com apelidos simultaneamente
Como acompanhar o ranking em tempo real
1. Na aba Simulados, clique em 🏆 Ranking ao Vivo no card da prova
2. O sistema busca os resultados do Firestore e exibe o ranking
3. A tela atualiza automaticamente a cada 30 segundos
4. Admin vê: apelido + nome real, acertos, tempo e posição
5. Alunos veem: apelido, acertos e porcentagem
6. O resultado do próprio aluno fica destacado em azul
7. Feche a tela para encerrar o polling
Como gerar e distribuir o PDF de uma prova
1. Na aba Simulados, localize a prova ou simulado desejado
2. Clique em 📄 PDF Prova para gerar a versão sem gabarito — envie para o aluno imprimir e resolver
3. Após o prazo, clique em 📄 PDF Gabarito para gerar a versão comentada — envie para o aluno corrigir
4. Os PDFs abrem automaticamente para impressão em nova aba
5. Dica: questões sem comentário aparecerão com aviso no PDF do gabarito — use "Exportar Questões" → "Sem comentário" para completar antes de distribuir
Como adicionar imagem a uma questão
1. No Banco de Questões, localize a questão com tabela/planilha/figura
2. Clique em ✏️ Editar
3. Role até o campo Imagem da Questão
4. Tire um print da tabela/figura e salve no computador
5. Clique em 📷 Selecionar imagem e escolha o print
6. A imagem é comprimida automaticamente e aparece no preview
7. Clique em 💾 Salvar — a imagem aparecerá antes do enunciado no modo TEC

Dica: Para questões com tabela no enunciado, tire o print só da tabela para economizar espaço.
Como importar uma prova anterior
1. Vá para a aba Importar → 📋 Prova Anterior
2. Preencha nome, banca, ano e cargo da prova
3. Cole o JSON ou selecione o arquivo
4. Clique em ✓ Validar e Importar
5. Acompanhe o progresso nas 3 etapas: Validar → Duplicatas → Salvar
6. Veja o resultado: novas criadas · vinculadas · similares para revisar
7. A prova aparece automaticamente em Provas Anteriores na aba Simulados
8. Se quiser transformar em simulado com cronômetro, clique em ⚙ Transformar no card
Como usar os 3 níveis de classificação
O sistema usa 3 níveis: Matéria → Tópico → Subtópico

Ao cadastrar ou editar uma questão:
1. Selecione a Matéria — o select de Tópico atualiza automaticamente
2. Selecione o Tópico — o select de Subtópico atualiza automaticamente
3. Selecione o Subtópico mais específico

Nos filtros do banco e modo TEC:
Use os 3 selects em cascata para encontrar exatamente as questões que quer estudar.

Questões antigas sem subtópico continuam funcionando normalmente — o campo é opcional.
Como mesclar matérias duplicadas
1. Vá em Configurações → Matérias & Tópicos
2. Localize a matéria que está duplicada (ex: "Lei 066/1993" com variação de nome)
3. Clique em ⛓ Mesclar na matéria que quer remover
4. Selecione a matéria de destino (a que vai ficar)
5. Confirme — todas as questões são movidas via updateDoc no Firestore
6. A matéria de origem é removida automaticamente
Como usar a Triagem de Desatualizadas
1. No Banco de Questões, clique em "Triagem Desatualizadas"
2. O sistema identifica candidatas por matéria + ano automaticamente
3. Confirme para exportar o JSON com as candidatas
4. Traga para o chat com Claude para análise
5. Reimporte as corrigidas com "Importar com Revisão"
